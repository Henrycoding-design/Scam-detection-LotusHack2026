Recommended setup order

Most teams get the best result by setting structure first, then importing active work, then customizing notifications and automation. None of these steps require outside tools, and every screen shown below lives inside the main workspace.

  1. Create team spaces for planning, delivery, and reporting.
  2. Invite collaborators and assign workspace roles.
  3. Import current tasks and map ownership fields.
  4. Review dashboard templates and publish your first report.
