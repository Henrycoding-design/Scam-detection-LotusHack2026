ScamShield Test Pages

These local fixtures are designed to exercise the extension's text, link, and meta heuristics without touching any real scam sites or collecting any real data.

Open the files directly for text and meta checks. If you want the current offsite-link and external-link-ratio heuristics to register, serve this folder from localhost instead of opening it via file://, because the scanner compares link hosts against the page hostname.
